# GoroBot
Go语言编写的跨平台聊天机器人框架

## [快速入门](https://jel1yspot.github.io/GoroBot/getting_started.html)

## 支持平台
- [x] QQ ([pkg/lagrange](https://github.com/Jel1ySpot/GoroBot/tree/master/pkg/lagrange))
- [ ] OneBot
- [ ] QQ Official
- [ ] Telegram

## 特性
- 高性能（我瞎说的反正 Go 怎么也比 Node 快）
- 高扩展性（插件化设计）
- 高自订性（插件想干啥都行.jpg）

## 注意事项
如果有能力，请**审查所有使用的代码！！！**
由于语言的特性，开发者拥有极高的权限，请在使用不可信来源的插件时多加注意！
